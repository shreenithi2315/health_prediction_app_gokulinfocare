# **Health prediction app by gokulinfocare**


I have created this app as part of my interview process.The application takes the patient's details including name,email,date of birth,cholestrol levels,glucose level and haemoglobin levels
.

## Technologies used 


1.Python

2.Streamlit

3.SQLite

4.Groq API

I have used SQLite3 for storing the datas since it is does not require seperate server setup and very simple commands. I used streamlit since i am familiar with it since i have used it for my past prediction dashboards and the layout is simple,neat and effective too.
I used Groq api since i found it easy to integrate compare to other APIs. I have created the duplicate env file for security purpose. Users can replace their API key for usage.

# App uses


Store the patients details from the blood test results with AI generated brief remarks. We can perform CRUD operations with the patients details and also can search the records with name.
